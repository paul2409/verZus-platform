# VERZUS Reference-Locked Visual System

Authority: approved mobile Rankings references, M2 visual language board, and Crew Icon Selector reference.

## Mandatory identity

- Canvas: near-black.
- Brand: cyan-to-white-to-violet VERZUS wordmark.
- Primary action/current player/success: neon green.
- Information and secondary action: cyan.
- Category/rank/special state: violet.
- Leadership and reward: gold/orange.
- Danger, loss, live urgency: red.
- Notification dots only: magenta.
- Shape language: clipped HUD corners, hard geometry, no soft SaaS rounding.
- Typography: Rajdhani for display and numeric emphasis; Inter for readable body text.
- Atmosphere: subtle grid, scanlines and restrained glow.

## Accessibility

Colour never acts as the only state indicator. Focus rings remain visible. Text contrast and reduced-motion behaviour remain mandatory.
