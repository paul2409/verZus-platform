# VERZUS Glossary

**Competition:** Organised event players or Crews can enter.

**Tournament:** A competition format using a bracket or staged progression.

**Season:** Time-bounded competitive cycle.

**Week:** Scoring and scheduling period inside a Season.

**Pool:** Group of players or Crews competing in a defined stage or weekly standing.

**Match:** One scheduled competitive contest.

**Fixture:** Scheduled Match within a competition structure.

**Check-in:** Time-bound confirmation that a participant is present and ready.

**Crew:** Persistent team-like organisation containing players across game lanes.

**Game lane:** A Crew or ranking track associated with one supported game.

**Crew War:** Multi-lane competition between Crews.

**Rank:** Current ordered position on a leaderboard.

**Rank movement:** Difference between current and previous ranking snapshot.

**Leaderboard:** Ordered ranking read model for a defined scope and ruleset.

**VS Points:** Competitive points used for ranking or progression as defined by a ruleset.

**REP:** Reputation or progression currency, subject to final economy design.

**Trust:** Explainable platform score based on verified behaviour and rule compliance.

**Division:** Skill or progression tier such as Rookie, Bronze, Silver, Gold, Elite, or Pro.

**Evidence:** Screenshot, recording, API result, or other artefact supporting a Match outcome.

**Dispute:** Formal review of a contested Match or Result.

**Result verification adapter:** Game-specific integration that converts external or manual evidence into a normalized verification decision.

**Pinned position:** A separate current-user ranking row shown even when the user is outside the visible top list.

**Stale data:** Previously valid data that may no longer be current.

**Feature flag:** Runtime control that enables a feature for selected environments or cohorts.

**Audit event:** Immutable record of a sensitive operation.

**Idempotency:** Property ensuring repeated identical commands do not create duplicate effects.
