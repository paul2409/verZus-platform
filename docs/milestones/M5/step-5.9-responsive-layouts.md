# Step 5.9 — Responsive Play Layouts

The Play Command Centre is verified at 360, 390, 430, 768, 1024, and 1440
pixels.

Rules:

- mobile prioritises the next match and check-in above secondary widgets
- tablet uses a single command column followed by paired secondary widgets
- desktop uses independent primary and secondary columns
- long handles, labels, numbers, and request IDs wrap safely
- no horizontal overflow is allowed
- reduced-motion preferences disable non-essential motion
