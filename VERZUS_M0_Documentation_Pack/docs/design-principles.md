# VERZUS Design Principles

## Visual direction
Premium retro-futurist esports interface with dark surfaces, bright high-contrast typography, disciplined neon accents, geometric HUD framing, and restrained texture.

## Hierarchy before decoration
The visual system must clearly distinguish:

- page title;
- current state;
- primary action;
- secondary controls;
- live data;
- supporting explanation.

## Brightness with restraint
Use bright accents for state, focus, rank, movement, success, and active navigation. Avoid applying glow to every element.

## Data should scan quickly
Players should identify rank, opponent, check-in status, time, points, streak, Crew, and action without reading long paragraphs.

## Tables should feel competitive
Avoid generic spreadsheet appearance. Use deliberate row emphasis, rank indicators, movement, identity cells, and state colour.

## Copy style
- Direct.
- Competitive.
- Clear.
- Minimal jargon.
- No fake urgency.
- No ambiguous button labels.

## Motion
Motion communicates:

- state transition;
- ranking movement;
- successful action;
- loading progress;
- attention requirement.

Reduced-motion mode must preserve meaning without animation.

## Accessibility
Visual identity must survive high contrast needs, colour blindness, keyboard operation, zoom, and screen readers. Colour never carries meaning alone.

## Image approval rule
Do not implement a full screen until its reference is approved. Approval confirms purpose, state, responsive anatomy, action priority, and build feasibility.
